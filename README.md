# Las-Vegas_Green_Space_Analysis

## Project Overview

This project analyzing the acessability to green spaces across the Las Vegas metropolitan area to identify neighborhoods tat. may lack adequate green space access

Urban heat and limited vegitation make green space and importan environmental and public health factor in desert cities

---


## Research Question

Which areas of Las Vegas have the lowest access to green space?

---

## Data Sources

U.S. Census Bureau

Clark County

---

## Methods

Spatial analysis techniques used:

- Nearest Distance Formula
- Reproject
- spatial joins
- raster processing

Python and GIS tools were used to automate and reproduce workflow

---

## Tools Used

Python
Jupyter Notebook

---

## Project Workflow

1. Import Spatial datasets
2. Generate distance to park
3. Identify underserved areas
4. Map Accessibility results

---

## Results

Key Findings include:

- Several neighborhoods in the East Las Vegasa area are more thab 1 mile from major parks.
- 1 neighborhood in North Las Vegas serves as the furthest distance from green spaces.
